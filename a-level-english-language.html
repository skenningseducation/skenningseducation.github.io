A Level English Language

Live, teacher-led lessons designed to build confident analysis and high-scoring exam responses. Suitable as supplementary support alongside school or as a main provision pathway.

Maximum class size: 10 Essay + data response technique Regular marked exam practice
Day 1 resources included: a complete set of lesson notes (Word/PDF), guided exam-style practice with model answers, and access to past papers for the student’s exam board.

What students will be able to do

  • Analyse spoken, written and multimodal texts accurately using appropriate terminology.
  • Write exam responses that meet assessment objectives: analysis, interpretation, structure and precision.
  • Handle data-driven questions (language variation, identity, change) and build well-supported arguments.
  • Plan and execute longer responses efficiently under timed conditions.

How lessons run

Weekly live teaching

  • Clear explanation + worked examples
  • Guided annotation and modelling
  • Targeted practice during the lesson
  • Frequent checks for understanding

Homework + feedback

  • Weekly practice tasks (short + extended)
  • Mark-scheme focused feedback
  • Redraft guidance so mistakes don’t repeat
  • Timed exam question practice throughout

Curriculum overview

English Language A Level varies slightly across exam boards, but the core skills and knowledge areas overlap strongly. The programme is delivered in a way that works across the main UK exam boards, with clear signposting for board-specific requirements.

Area Focus
Text analysis How to analyse language choices, structure, register, audience, purpose, and representation in written/spoken texts.
Language frameworks Core terminology and methods for analysing lexis, grammar, semantics, pragmatics, discourse, and phonology (as required).
Language and identity / society How language reflects identity, power, occupation, gender, age, region; evaluating viewpoints using evidence.
Language variation and change Key concepts and how to handle data: comparing texts, explaining differences, interpreting patterns and trends.
Writing skills How to craft high-mark responses: introduction, thesis, paragraph structure, evidence selection, evaluation, and conclusions.
Exam technique Timing, planning, question decoding, meeting assessment objectives, and avoiding common pitfalls.

36-week outline

A practical week-by-week plan (adapted to the student’s exam board and class pace).

Weeks Focus Typical outcomes
1–6 Foundations of analysis + core terminology Confident annotation; accurate terminology; structured paragraphs using evidence.
7–12 Comparative analysis (two texts / viewpoints) Comparison techniques; clear argument; well-chosen quotations; stronger evaluation.
13–18 Language & identity / representation Analysing how language constructs identity; linking to context and purpose; stronger conclusions.
19–24 Variation + data response skills Handling datasets; interpreting patterns; writing evidence-led explanations and evaluations.
25–30 Long-form responses + independent planning Faster planning; stronger structure; improved coherence and precision under timed conditions.
31–36 Revision + exam mastery Timed papers; examiner-style marking; personalised improvement targets and final polish.
